📖 API 文档参考

基本属性

title: 通知标题

body: 通知内容

icon: 图标URL

image: 图片URL

badge: 徽章图标URL

行为属性

tag: 通知标签（用于替换）

requireInteraction: 需要用户交互

renotify: 重新通知时振动

silent: 静音模式

vibrate: 振动模式数组

事件处理

onclick: 点击事件

onclose: 关闭事件

onshow: 显示事件

onerror: 错误事件
