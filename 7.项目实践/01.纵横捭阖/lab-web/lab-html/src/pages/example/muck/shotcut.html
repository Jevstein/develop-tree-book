关键属性对比

事件属性 说明 Shift+1 返回值
event.key 按下键的字符值（考虑修饰键） "!"
event.code 按下键的物理键值（忽略修饰键） "Digit1"
event.keyCode 弃用 - 物理键码值 49

推荐方案： 使用 event.code 属性检测物理按键位置

示例：处理"Shift+1"按键事件，获取实际物理键"1"

document.getElementById('input').addEventListener('keydown', function(e) {

if (e.code === 'Digit1') { // 物理按键检测

console.log('物理键1被按下');

}
