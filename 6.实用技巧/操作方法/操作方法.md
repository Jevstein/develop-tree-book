# 操作方法

## 1、windows上查看端口被谁占用

如：排查是谁占用了80端口

* 1.打开cmd，找到pid

```bat
$ netstat -aon | findstr "80" 
TCP   127.0.0.1:80     0.0.0.0:0        LISTENING    2448
```

* 2.查看任务管理器，定位pid

  任务管理器 => 选择菜单查看 => 选择列 => 勾选PID

  

## 2、mkcert 生成并信任 HTTPS 证书

`mkcert` 是一个零配置工具，它生成的证书会被 Chrome、Safari、Firefox 等主流浏览器直接信任，无需手动处理安全警告。

以下在mac系统操作：

```bash
#1、安装 mkcert：确保已安装 Homebrew，然后在终端执行
$ brew install mkcert
# 如果需要使用 Firefox，还需安装 nss：
$ brew install nss

#2、安装本地 CA：执行以下命令，它会在系统中安装一个本地根证书颁发机构
$ mkcert -install
此步骤会将根证书添加到 macOS 的系统钥匙串中，并设置为受信任。

#3、生成证书：为你的域名（如 localhost 或自定义域名 dev.example.com）生成证书和私钥：
$ mkcert localhost 127.0.0.1 ::1
将生成两个文件：
localhost+3.pem：证书文件
localhost+3-key.pem：私钥文件

#4、配置 Web 服务器：在Web服务器（如 Nginx、Apache、Node.js 等）中配置 SSL，指向 .pem 和 -key.pem
server {
    listen 443 ssl;
    server_name localhost;
    ssl_certificate     /path/to/localhost+3.pem;
    ssl_certificate_key /path/to/localhost+3-key.pem;
    # 其他配置...
}
```

也可以使用 OpenSSL 手动生成并信任（传统方法）

```bash
$ openssl req -x509 -newkey rsa:2048 -keyout key.pem -out cert.pem -days 365 -nodes -subj "/CN=localhost" -addext "subjectAltName = DNS:localhost, IP:127.0.0.1"

将生成 key.pem（私钥）和 cert.pem（证书）
```
