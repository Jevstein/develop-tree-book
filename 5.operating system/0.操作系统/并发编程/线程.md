## 线程安全
* 安全  
    * 无状态：无全局变量
        ```
        void thread_proc(int j)
        {
            int i=1;
            j = j + 1;
        }
        ```
    * 原子操作
    * 加锁
* 不安全
    * 有状态：有全局变量
        ```
        void thread_proc(int j)
        {
            count++；
            int i=1;
            j = j + 1;
        }
        ```
