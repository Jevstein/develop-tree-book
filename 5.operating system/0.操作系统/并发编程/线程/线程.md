##  2. <a name='-1'></a>线程安全
###  2.1. <a name='-1'></a>安全  
* 无状态：无全局变量
    ```
    void thread_proc(int j)
    {
        int i=1;
        j = j + 1;
    }
    ```
* 原子操作
* 加锁
###  2.2. <a name='-1'></a>不安全
* 有状态：有全局变量
    ```
    void thread_proc(int j)
     {
         count++；
        int i=1;
        j = j + 1;
    }
     ```
